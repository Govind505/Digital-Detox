# Privacy Policy Draft
We collect only the minimum data necessary to provide focus coaching features.

- Local data: session history, streak, and XP (stored on-device).
- Optional Firebase Analytics/Crashlytics: app performance and aggregated events.
- Optional Usage Access permission: app usage durations and unlock trends read from Android and shown in-app.

Permissions are optional and can be denied without breaking core focus functionality.
No external payment links are used.
